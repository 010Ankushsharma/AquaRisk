# 🌊 AquaRisk Score  
### AI-Powered Water Contamination Severity Prediction System

---

## 🚀 Overview  
AquaRisk Score is an AI-driven system that predicts **water contamination severity (0–100)** using environmental, water quality, and public health data.  
It enables **early warning for water-borne diseases** in rural areas by providing real-time risk insights and actionable alerts.

---

## 🎯 Problem Statement  
Rural regions often face delayed detection of contaminated water, leading to outbreaks of diseases like diarrhea, cholera, and typhoid...

This project solves that by:  
- Monitoring multiple risk factors  
- Predicting contamination severity using ML  
- Enabling proactive decision-making  ..

---

## 🧠 Solution  
We built a **regression-based machine learning model** that outputs a **continuous contamination score (0–100)** instead of simple safe/unsafe classification.

### 🔥 Why this is unique:
- Continuous risk scoring (more precise)
- Combines **water + weather + health data**
- Real-time prediction capability
- Explainable insights (why risk is high)
